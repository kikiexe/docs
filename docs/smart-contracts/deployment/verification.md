# Contract Verification

All Veilfi contracts are verified on Mantle Sepolia Explorer for transparency.

## Verification Process

```bash
npx hardhat verify --network mantleSepolia \
  <CONTRACT_ADDRESS> \
  <CONSTRUCTOR_ARGS>
```
